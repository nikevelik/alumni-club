# alumni-club

## Pages

**reg.html — Registration page**
Enables a new user to create an account and gain access to the system.

**log.html — Login page**
Allows an existing user to authenticate and enter the system.

**dir.html — Directory page**
Lets the user find and access other alumni.

**vie.html — Profile page (view)**
Allows the user to view information about a specific alumnus.

**edi.html — Edit profile page**
Enables the user to update their own information.
